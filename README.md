# Expense Tracker

A web-based Expense Tracker developed using Python Flask and SQLite to manage daily income and expenses efficiently.

## Features

- Add income and expense transactions
- Categorize transactions
- View transaction history
- Calculate balance automatically
- User login system
- Responsive UI

## Technologies Used

- Python
- Flask
- SQLite3
- HTML
- CSS
- JavaScript
- Pandas

## Project Structure

expense-tracker/
│── app.py
│── database.db
│── requirements.txt
│── templates/
│── static/

## How to Run

1. Install dependencies

pip install -r requirements.txt

2. Run project

python app.py

3. Open browser

3. Open browser locally after running Flask server

http://127.0.0.1:5000 (Localhost only)
